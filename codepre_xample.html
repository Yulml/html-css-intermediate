Two CSS examples

In the center of the window you can see the CSS for a class that sets an element centered both horizontally and vertically. 

<style>
  .vhcenter { 
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
  }
</style>

And right below this line you can see an example on how to apply changes to even and odd elements of a tr inside whatever container has the class "blueish". 

<style>
  .blueish tr:nth-child(even) {
  background-color: #f6f6f6;
  }
         
  .blueish tr:nth-child(odd) {
  background-color: #eeeeee;
  }
</style>
