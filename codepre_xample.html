Two CSS examples

This is a css class you can include in your header so you can centre both horizontally and vertically.

<style>
  .vhcenter { 
  position: absolute;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
  }
</style>

And this is an example on how to apply changes to even and odd elements of tr

<style>
  .blueish tr:nth-child(even) {
  background-color: #f6f6f6;
  }
         
  .blueish tr:nth-child(odd) {
  background-color: #eeeeee;
  }
</style>
